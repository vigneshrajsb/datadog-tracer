Note: The application with the domain should be instrumented for Application Performance Monitoring (APM)

Instructions

1. Instrument Datadog APM for your frontend and services. Read Datadog docs

2. Add your domain to Settings and Save

3. Navigate to your instumented web application, open extension and click Toggle

4. The extension generates a Trace ID and injects required headers to network requests matching the domain pattern


Note: The goal of this extension is to create a Trace without thinking about retention filters for debugging purposes only.
